## 📌 What It Does
HONQPGen automatically generates questions from any educational
paragraph using NLP and Deep Learning.
No manual effort. Just paste and generate.

---

## ⚙️ How It Works
```
Paste Text → SpaCy extracts keywords → T5 AI generates questions → Done!
```

---

## 🛠️ Tech Stack
| Tool | Purpose |
|---|---|
| Python | Core language |
| SpaCy | Keyword extraction |
| T5 (HuggingFace) | Question generation |
| Streamlit | Web UI |
| PyTorch | Deep learning backend |
